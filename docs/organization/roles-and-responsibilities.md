# Roles and Responsibilities

This document defines the responsibilities, ownership boundaries, and expected activities for each role participating in ERP Plus.

---

## Product Owner

### Responsibilities

- Own product vision
- Define roadmap
- Manage Epics
- Create and prioritize User Stories
- Define acceptance criteria
- Approve completed functionality
- Participate in Sprint Planning
- Participate in Sprint Review

### Primary Tools

- Taiga
- ERP Plus Documentation

→ [Product & Delivery Management](../product/index.md)

→ [Taiga Workflow](../product/taiga-workflow.md)

→ [Sprint Process](../product/sprint-process.md)

→ [Release Management](../product/release-management.md)
