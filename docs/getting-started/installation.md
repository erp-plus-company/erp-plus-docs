# Installation

## Requirements

- Ruby 3.3+
- Rails 8.0+
- PostgreSQL 16+
- Docker (optional)

---

## Clone Repository

```bash
git clone --recurse-submodules git@github.com:erp-plus-company/erp_plus.git

cd erp_plus
```

## Install Dependencies

```bash
bundle install
```

## Setup Environment

```bash
bin/setup
```

## Start Application

```bash
bin/dev
```
