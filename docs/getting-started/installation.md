# Installation

## Requirements

- Ruby 3.3+
- PostgreSQL 16+
- Node.js
- Yarn
- Docker (optional)

---

## Setup

```bash
git clone --recurse-submodules git@github.com:erp-plus-company/erp_plus.git

cd erp_plus

bundle install

bin/setup
```

---

## Run the application

```bash
bin/dev
```

## Verify

Visit:

[http://localhost:3000](http://localhost:3000)
