# Getting Started

Welcome to ERP Plus documentation.

This guide will help you understand how to install, run, and contribute to the platform.

---

## What is ERP Plus?

ERP Plus is a modular multi-tenant ERP system built with:

- Rails 8
- Modular Engines architecture
- Docker + Kamal deployment
- GitHub CI/CD pipelines

---

## Key Concepts

- Modular monolith design
- Engine isolation
- Multi-tenant architecture
- CI/CD automation
